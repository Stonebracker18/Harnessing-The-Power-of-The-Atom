# Harnessing the Power of the Atom

## Overview
This project explores advanced nuclear energy technologies through the design of an Advanced Fast Breeder Thorium Reactor (AFBTR).

## Features
- AFBTR reactor concept
- Fuel breeding process
- Gravity-driven emergency cooling
- Molten salt backup drain tank
- Interactive reactor simulator
